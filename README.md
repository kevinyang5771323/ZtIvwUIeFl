# 项目简介 python337

本项目是一个基于爬虫技术的个性化书籍推荐系统，采用 Python 语言进行开发，以 Django 框架为基础搭建后端服务，前端则使用了 jQuery、Bootstrap 等技术栈。

## 技术栈

### 后端

- **语言与框架**：Python 3.x，Django
- **数据库**：MySQL 5.7

### 前端

- jQuery
- Bootstrap
- HTML
- JSP

### 开发工具

- PyCharm

## 功能模块

- 数据爬取：从指定的资源中抓取书籍信息
- 数据处理：清洗、整合爬取到的书籍数据
- 个性化推荐：根据用户行为生成个性化书籍推荐
- 数据可视化：将书籍数据和用户行为以可视化形式展现

## 系统角色

- 系统管理员：负责系统的维护和监控
- 普通用户：获取个性化书籍推荐，进行书籍浏览

## 运行环境

- 操作系统：兼容性良好，无特殊要求
- 后端：Python 3.x，MySQL 5.7
- 前端：现代浏览器

## 部署步骤

1. 环境准备：安装 Python 3.x，MySQL 5.7，PyCharm
2. 数据库搭建：按照 schema 导入数据库结构
3. 后端部署：配置 Django 项目，部署至服务器
4. 前端部署：将前端页面放置于 Web 服务器目录下
